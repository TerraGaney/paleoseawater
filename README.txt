Supplemental data for 'Composition, properties, speciation, and pH of paleo-seawaters.' 

Contact: Terra Ganey, tganey@ucsc.edu; Mathis Hain, mhain@ucsc.edu

Reference: Ganey, T.M., Hain, M.P., Clegg, S.L., Turner, D.R., & Paytan, A. (2026). Composition, properties, speciation, and pH of paleo-seawaters. [Dataset].

––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––
CONTENTS:

(1) folder 'look-up table', which contains a copy of the MarChemSpec-based K look-up table for non-modern seawater compositions, plus instructions for calculating equilibrium constants (Ks) using table contents
(2) folder 'MarChemSpec-output-processing', which contains data and scripts used for analysis in the manuscript

Please see separate README files in each folder for detailed instructions.
